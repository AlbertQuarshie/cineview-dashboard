# CineView Dashboard

This project is the creation of a Movie Dashboard interface using only HTML and CSS.

## Methods Used

### A. Layout and Structure

1. Created a fixed top navigation bar with logo, search bar, and profile picture.
2. Added a fixed sidebar navigation with links for Home, Trending, and Library.
3. Built a responsive main content area to display movie cards in a flexible grid.

### B. Typography

1. Added 3 fonts from Google Fonts (`Monoton`, `Source Sans 3`, `Sancreek`) using embed code.
2. Applied font sizes and styles to distinguish headings, logo, and body text.
3. Ensured readability with proper spacing and alignment.

### C. Movie Cards & Glass Effect

1. Designed movie cards with rounded corners, padding, and a subtle backdrop blur.
2. Added hover effect on images to indicate interactivity.
3. Layered elements using flexbox for alignment and spacing.

### D. Colors and Theme

1. Used soft green tones (`#74c69d`) for top navigation and main content background.
2. Used dark green (`#1b4332`) for sidebar and movie card backgrounds.
3. Applied beige text color for contrast and readability.

### E. Footer

1. Added a footer with centered copyright.
2. Matched the footer color with the main theme for consistency.
